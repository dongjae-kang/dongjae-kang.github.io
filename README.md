# Dongjae (Jack) Kang Portfolio

Personal portfolio website built with React, Vite, styled-components, framer-motion, D3, and HashRouter for GitHub Pages.

## Scripts

- `npm run dev`
- `npm run build`
- `npm run preview`
- `npm run deploy`

## Deployment

This repository is configured for GitHub Pages deployment via `gh-pages`, publishing the `dist/` directory.
